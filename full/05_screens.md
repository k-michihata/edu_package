# 05 画面構成

## 画面一覧

### 生徒側

| ID | 画面名 | 説明 |
|----|--------|------|
| S-01 | ログイン画面 | 学校アカウントでログインする |
| S-02 | ホーム画面 | 現在のフェーズと進捗を確認し、続きから再開できる |
| S-03 | 価値観記述画面 | 教師が設定した問いかけに対してテキストで記述する |
| S-04 | キャリアアンカー診断画面 | 8項目の診断設問に回答する |
| S-05 | 診断結果画面 | 行動タイプの診断結果を確認する |
| S-06 | 将来事象入力画面 | カテゴリ別に「●年後に○○」形式で3〜5個入力する |
| S-07 | シミュレーション画面 | AIが提示した事象に対してポジ/ネガ評価と行動記述を行う |
| S-08 | 人生グラフ画面 | シミュレーション結果をもとに生成された人生グラフを確認する |

### 教師（管理者）側

| ID | 画面名 | 説明 |
|----|--------|------|
| A-01 | アドミンホーム画面 | クラス・生徒の進捗を一覧で確認する |
| A-02 | 問いかけ編集画面 | 価値観記述フェーズの問いかけを編集・保存する |
| A-03 | 人生グラフ一覧画面 | 生徒全員の人生グラフを一覧で確認する |

## 各画面の詳細

### S-01 ログイン画面
- 学校アカウント（Google Workspace等）でのSSOログインボタン
- ログイン後、前回の続きがある場合はホーム画面へ自動遷移

### S-02 ホーム画面
- 現在のフェーズ（自己分析 / シミュレーション前半 / 後半 / 完了）を表示
- 進捗状況のステップ表示（例：ステップ1〜5）
- 「続きから始める」ボタン

### S-03 価値観記述画面
- 教師がカスタマイズした問いかけを表示
- テキストエリアで自由記述
- 複数の問いかけがある場合は1問ずつ表示、または一覧表示（設計時に検討）

### S-04 キャリアアンカー診断画面
- 高校生向けに読み替えた8項目に対応する設問を表示
- 回答形式：選択式（例：5段階評価）または順位付け（要検討）
- 進捗バー表示

### S-05 診断結果画面
- 行動タイプ（キャリアアンカー）の結果を表示
- 各アンカーのスコアをグラフ等で可視化
- 結果の簡単な説明文を表示
- 「次へ（事象入力へ）」ボタン

### S-06 将来事象入力画面
- カテゴリ選択（社会変化 / ライフイベント など）
- 「●年後に○○が起こる」形式の入力フォーム
- 3〜5個入力できる（最低3個・最大5個のバリデーション）
- 年代のばらつきをガイドするヒント表示

### S-07 シミュレーション画面
- AIが生成した事象を1つずつ表示（全5個）
- 各事象に対して：
  - ポジティブ / ネガティブ の評価ボタン
  - 行動記述のテキストエリア
- 進捗表示（例：1/5）
- 前の事象に戻る機能（要検討）

### S-08 人生グラフ画面
- 横軸：年齢・ライフステージ（10代〜80代）
- 縦軸：感情のポジ（＋）/ ネガ（ー）
- 各事象のポイントを線で結んだグラフ
- 各ポイントに事象名・評価・記述内容をホバー/タップで表示
- 拡張：価値観領域マップを別タブ等で表示

### A-01 アドミンホーム画面
- クラス一覧・生徒一覧の表示
- 各生徒の現在のフェーズ・進捗状況を一覧で確認
- フェーズ別のフィルタリング機能

### A-02 問いかけ編集画面
- 現在設定されている問いかけの一覧表示
- 問いかけの追加・編集・削除
- 保存・反映ボタン

### A-03 人生グラフ一覧画面
- 生徒全員の人生グラフを一覧表示
- 個別の生徒グラフを拡大表示する機能
- クラス・グループでのフィルタリング機能
