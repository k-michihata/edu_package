# 02 ユーザー定義

## ユーザー種別

| 種別 | 説明 |
|------|------|
| 生徒 | ツールのメインユーザー。自己分析・シミュレーションを行う |
| 教師（管理者） | 授業を設計・管理する。アドミン画面を使用する |

---

## ペルソナ

### 生徒

- **年齢：** 15〜18歳（中高生、主に高校生）
- **利用場面：** キャリア教育の授業内
- **利用形態：** 個人で操作するが、事象のブレインストーミングはグループで実施
- **ITリテラシー：** スマートフォン・タブレットの日常的な使用経験あり。ただし学習ツールとしてのPC操作は学校環境に依存する
- **モチベーション：** 授業の一環として取り組むため、主体的な動機づけには工夫が必要
- **課題：** 将来について考える経験が少なく、職業・私生活の具体的なイメージを持ちにくい

### 教師（管理者）

- **役割：** キャリア教育の担当教師
- **利用場面：** 授業設計・進捗管理・振り返り指導
- **ITリテラシー：** 基本的なPC操作は可能。専門的な設定は避けたい
- **ニーズ：**
  - 生徒の進捗をひと目で把握したい
  - 問いかけの内容を自分の授業に合わせてカスタマイズしたい
  - 生徒の人生グラフを一覧で確認し、授業の振り返りに活用したい

---

## ユースケース

### 生徒

| # | ユースケース | 説明 |
|---|-------------|------|
| U-01 | ログイン | 学校アカウントでログインし、前回の続きから再開できる |
| U-02 | 価値観の記述 | 教師がカスタマイズした問いかけに対して自分の価値観をテキストで記述する |
| U-03 | 行動タイプ診断 | キャリアアンカーベースの診断（高校生向け）を受け、自分の行動タイプを把握する |
| U-04 | 将来事象の入力 | グループワークで出た事象の中から個人で3〜5個を選び「●年後に○○が起こる」形式で入力する |
| U-05 | シミュレーション（前半） | 10〜40代の事象5個に対してポジ/ネガ評価と行動記述を行う |
| U-06 | シミュレーション（後半） | 50〜80代の事象5個に対してポジ/ネガ評価と行動記述を行う |
| U-07 | 人生グラフの確認 | シミュレーション結果をもとに生成された人生グラフを確認する |

### 教師（管理者）

| # | ユースケース | 説明 |
|---|-------------|------|
| A-01 | 進捗確認 | クラス・生徒ごとの進捗状況を一覧で確認する |
| A-02 | 問いかけカスタマイズ | 価値観記述フェーズの問いかけ内容を編集・保存する |
| A-03 | 人生グラフ一覧 | 生徒全員の人生グラフを一覧で確認し、授業の振り返りに活用する |

---

## 認証

| 項目 | 内容 |
|------|------|
| ログイン方式 | 学校アカウント（例：Google Workspace for Education）によるSSO |
| データ継続性 | ログインすることで前回の操作状態を引き継いで再開できる |
| 対象地域 | 日本 |
