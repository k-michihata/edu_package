# 00 MVP仕様書

## プロジェクト概要

将来可能性教育を支援するITツール。
高校生が将来起こりうる事象に対して感情・行動を考え、人生グラフとして可視化することで自己理解を深める。

> 全体像は `docs/full/` を参照。

---

## MVPのゴール

- AIが生成した将来の事象に対して、ポジ/ネガ評価と行動記述を行う
- シミュレーション結果を人生グラフとして表示する
- アカウント・ログイン不要。ブラウザ上で完結する

---

## MVPスコープ

### 含む機能

| # | 機能 | 説明 |
|---|------|------|
| 1 | AI事象生成（前半） | 10〜40代で起こりうる平均的な事象を5個生成する |
| 2 | AI事象生成（後半） | 50〜80代で起こりうる平均的な事象を5個生成する |
| 3 | ポジ/ネガ評価 | 各事象に対してポジティブ/ネガティブを選択する |
| 4 | 行動記述 | 各事象に対して自分の行動をテキストで記述する |
| 5 | 人生グラフ表示 | シミュレーション結果を人生グラフとして表示する |

### 含まない機能（将来対応）

| # | 機能 |
|---|------|
| - | ログイン・アカウント管理 |
| - | セッション継続（データ永続保存） |
| - | 自己分析フェーズ（価値観記述・キャリアアンカー診断） |
| - | 生徒による事象入力 |
| - | 行動タイプ別の事象優先順位調整 |
| - | アドミン機能（教師向け） |
| - | 価値観領域マップ |

## ユーザーフロー

```
① トップ画面
   └ 「シミュレーションを始める」ボタン
         ↓
② シミュレーション（前半：10〜40代）
   └ AIが事象を5個生成
   └ 1事象ずつ表示
       └ ポジ/ネガ評価を選択
       └ 行動をテキストで記述
       └「次へ」で次の事象へ
         ↓
③ シミュレーション（後半：50〜80代）
   └ 同上（5個）
         ↓
④ 人生グラフ表示
   └ 横軸：年齢・ライフステージ（10代〜80代）
   └ 縦軸：感情のポジ（＋）/ ネガ（ー）
   └ 各事象をポイントとしてプロット・線で結ぶ
   └ 各ポイントに事象名を表示
```

## 画面構成

| 画面 | 説明 |
|------|------|
| トップ画面 | 開始ボタンのみのシンプルな画面 |
| シミュレーション画面 | 事象を1つずつ表示・評価・記述する画面 |
| 人生グラフ画面 | シミュレーション結果を可視化する画面 |

## データ設計（MVP）

### セッション内で保持するデータ（ブラウザメモリ）

```javascript
{
  phase: "first" | "second",
  answers: [
    {
      event_id: string,
      age_at_event: number,        // 事象発生時の年齢
      description: string,         // 事象の内容
      evaluation: "positive" | "negative",
      action_text: string
    }
  ]
}
```

### AI事象生成の入出力

**入力（プロンプトに渡すもの）**
- 対象年代：前半（10〜40代）/ 後半（50〜80代）
- 生成する事象数：5個

**出力（AIに返してもらうもの）**
```json
[
  {
    "age_at_event": 22,
    "description": "大学を卒業し、就職活動を経て社会人になる"
  }
]
```

---

## AI事象生成のプロンプト方針

- 日本の高校生を主語とした、平均的なライフイベント・社会変化を生成する
- ポジティブ・ネガティブな事象を混在させる
- 社会変化とライフイベントを混在させる
- 対象年代内で年齢がばらけるように生成する
- 出力はJSON形式のみ（余分なテキスト不要）

---

## 人生グラフの仕様

| 項目 | 仕様 |
|------|------|
| 横軸 | 年齢（10代〜80代） |
| 縦軸 | ポジ（＋1）/ ネガ（ー1） |
| プロット | 各事象を点でプロット |
| 線 | 点を時系列順に結ぶ |
| ラベル | 各点に事象名を表示 |
| ライブラリ候補 | Recharts / Chart.js |
